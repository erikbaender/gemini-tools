# Privacy Policy for Gemini Tools

Last updated: 2026-04-16

## Overview
Gemini Tools has the purpose of improving the user experience of the Gemini web app by reducing common pain points like the mode getting switched randomly when opening a new chat. It is designed to process data locally in your browser.

## Data We Collect
Gemini Tools does not collect personal data for transmission to a server.

The extension stores only local preference data in Chrome extension storage (`chrome.storage.local`), such as:
- Whether model correction is enabled
- Whether correction notifications are enabled
- Whether the upgrade button should be hidden
- The last manually selected Gemini model

## How Data Is Used
Stored local preferences are used only to provide the extension's core functionality:
- Remember your selected settings
- Restore your selected model when Gemini switches it automatically
- Apply optional interface behavior you enabled

## Data Sharing
Gemini Tools does not sell, transfer, or share user data with third parties.

## Remote Code and Tracking
Gemini Tools does not use remote executable code.

Gemini Tools does not include analytics, advertising trackers, or telemetry endpoints.

## Data Retention and Control
Data is stored locally in your browser and remains until you:
- Change settings inside the extension
- Clear browser extension storage
- Uninstall the extension

## Permissions
Gemini Tools requests only the permissions needed for its single purpose:
- `storage`: to persist local user preferences
- Gemini host access via content script match patterns: to read and adjust model selection behavior on Gemini pages only

## Children's Privacy
Gemini Tools is not directed to children and is not intended to collect personal information from children.

## Changes to This Policy
This policy may be updated to reflect extension changes or legal requirements. The latest version will be available at this location.
